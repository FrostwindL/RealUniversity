About

Pre-Registration for enrollment as well as easilly accessible list of offered Courses and Subjects

RealUniversity API

This API allows your to :

  • Pre Register your credentials and course
  • See the full list of available courses and subjects
  • Check specific courses and subject

Endpoints


The API is available at https:/RealUniversity

Status


GET /test

Check the API status


List of courses

GET /courses

Returns a list of available courses


Get a single course

GET /courses/:id

Returns detailed info on a specific course


List of students

GET /students

Returns a list of available courses


Submit pre enrollment

POST /students

Allows you to submit a pre enrollment entry

The request body needs to be in JSON format including the following properties

  • name - String - Required
  • course_id - Integer - Required
  • year - String - Required
  • contact - String - Required
  • email - String - Required

Example

Post /students/
{
"name" : "Lastname Firstname"
"course_id" : 1
"year" : "2nd"
"contact" : "1234567890"
"email" : "Enrolling_Student@gmail.com"
